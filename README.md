## 📖 Sobre o Projeto

GitHelp é uma aplicação web interativa que **revoluciona** a forma como você interage com Git. Desenvolvida especialmente para a comunidade brasileira de desenvolvedores, nossa plataforma permite que você obtenha comandos Git precisos usando **linguagem natural em português**.

### 🎯 Problema Resolvido
- ❌ Dificuldade em memorizar sintaxes complexas do Git
- ❌ Barreiras linguísticas em documentações técnicas
- ❌ Falta de recursos educacionais em português
- ❌ Interface intimidadora para iniciantes

### ✅ Nossa Solução
- ✨ **Interface 100% em português** com explicações claras
- 🤖 **Assistente inteligente** que compreende linguagem natural
- 📚 **Tutoriais progressivos** do básico ao avançado  
- 🌓 **Experiência personalizada** com temas claro/escuro

## 🚀 Instalação e Uso

### 🌐 Uso Online (Recomendado)
Acesse diretamente em: **[https://githelp.com.br](https://githelp.com.br)**

### 💻 Instalação Local

```bash
# Clone o repositório
git clone https://github.com/inojoza28/githelp.git

# Navegue para o diretório
cd githelp

# Abra no navegador
# Linux
xdg-open index.html

# macOS  
open index.html

# Windows
start index.html
```

### 📋 Requisitos
- Navegador moderno (Chrome 80+, Firefox 75+, Safari 13+, Edge 80+)
- JavaScript habilitado
- Conexão com internet (para CDNs)

## 🎮 Como Usar

### 1️⃣ **Busca de Comandos**
```
Digite: "quero enviar mudanças para o github"
Resultado: git push origin main
```

### 2️⃣ **Tutoriais Progressivos**
- Selecione seu nível (Iniciante → Avançado)
- Siga os passos interativos
- Pratique com exemplos reais

### 3️⃣ **Atalhos de Produtividade**
- `Alt + T`: Alternar tema
- `Ctrl + /`: Focar na busca
- `Esc`: Fechar modais

Sua contribuição é **muito bem-vinda**! Veja como participar:

1. **Fork** o projeto
